1. Goplements.nvim                           *goplements.nvim-goplements.nvim*

Goplements is a small neovim plugin for Go: brings implementations into the
foreground for structs and interfaces.

ðŸš€ USAGE AND API        *goplements.nvim-goplements.nvim-ðŸš€-usage-and-api*

The plugin is automatically enabled for Go files using an autocmd.

Three commands are provided for disabling, enabling and toggling the plugin:

- `:GoplementEnable`
- `:GoplementDisable`
- `:GoplementToggle`

The same commands are also available as lua functions:

- `require("goplements").enable()`
- `require("goplements").disable()`
- `require("goplements").toggle()`
